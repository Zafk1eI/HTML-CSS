Да, поперечная ось в CSS флекс-боксе может быть направлена снизу вверх.
Если свойство flex-direction имеет значение row, то главная ось будет направлена горизонтально, слева направо. Поперечная ось при этом будет вертикальной, направлена сверху вниз.
Если свойство flex-direction имеет значение column, то главная ось будет направлена вертикально, сверху вниз. Поперечная ось при этом будет горизонтальной, направлена слева направо.
Если свойство flex-direction имеет значение row-reverse, то главная ось будет направлена горизонтально, но в этом случае она будет направлена справа налево. Поперечная ось при этом останется вертикальной и будет направлена сверху вниз.
Если свойство flex-direction имеет значение column-reverse, то главная ось будет направлена вертикально, но в этом случае она будет направлена вверх. Поперечная ось при этом останется горизонтальной и будет направлена слева направо.
